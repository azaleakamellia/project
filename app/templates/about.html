Project Overview

This is the about page outlining the project. It aims to be concise with information pertaining the project and strives to maintain a clean and simple interface. The project is still in its early stages and is still a work in progress. Any effort to collect and centralize information should not always employ big investment with minimal effort at our end to be commited to the technicalities of the endeavour.

How is Utility Map made?

The application utilizes the cacophony of the following tools and services:

  • PostgreSQL/PostGIS: Data storage and management
  • Python: Geospatial libraries for data manipulation and visualization
  • Leaflet.js: Interactive map rendering
  • GeoDjango: Web Framework
